# Identica - Студенческий портал

Django приложение для управления профилями студентов и подписками на сайты.

## Функции
- Профили студентов
- Система подписок на сайты
- Админ-панель для управления

## Установка
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
